github pages的理解和使用心得

我們可以建立/docs目錄，並在此目錄下建立靜態網頁(如default index.html)，並且指定github pages設定在/docs目錄下，

目前只允許/和/docs目錄下建立靜態網頁，我覺得設定在/docs目錄下是比較合適的方法，

如下是試用的結果

https://github.com/charlie388/test-repo-pages-1

https://charlie388.github.io/test-repo-pages-1/



如果希望/目錄下建立靜態網頁，可在此目錄下建立靜態網頁(如default index.html)，並建立gh-pages branch，

系統會自動設定好gh-pages branch 的 /目錄為github pages的靜態網頁位址，

想不到建立gh-pages branch的優點...，除非/docs目錄對開發者有特殊需求

如下是試用的結果
